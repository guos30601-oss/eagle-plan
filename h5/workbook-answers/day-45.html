#题型答案解析
01语法填空Had考查虚拟条件句的倒装。原句为 If you had come a little earlier，省略 if 之后将助动词 Had 提至句首并大写。
02语法填空Should考查虚拟条件句的倒装。原句为 If it should rain tomorrow，省略 if 之后将情态动词 Should 提至句首并大写。
03语法填空would be考查错综时间条件句。从句表示对过去（in the past）的假设，用过去完成时；主句表示对现在（now）的假设，谓语用 would + 动词原形。
04语法填空would not come考查错综时间条件句。从句是对过去的假设（had informed），主句是对现在（at this moment）的假设，形式用 would not + 动词原形。
05语法填空would have sent考查错综时间条件句。从句是对现在情况的假设（were free），主句是对过去情况（yesterday）的假设，主句用 would have + 过去分词 形式。
06语法填空would have greeted考查错综时间条件句。从句是对现在情况的假设（knew），主句是对过去（the other day）的假设，主句形式用 would have done。
07语法填空would regret / might regret考查与将来事实相反的主句。从句用一般过去时，主句对应使用 would/might/could + 动词原形。
08语法填空Had考查省略 if 的倒装。原句为 If the governments and scientists had not worked together，省略 if 之后将第一个助动词 Had 提至句首。
09语法填空had not given考查与过去事实相反的条件从句。由主句的 might have failed 和时间词 then 可知是对过去虚拟，从句用过去完成时 had not given。
10语法填空had rained考查与过去事实相反的虚拟。根据时间词 yesterday 和主句 would have been 判定，从句用过去完成时 had rained。
11单项选择B考查宾语从句虚拟语气 。 advise （建议）后面的宾语从句 ，谓语动词采用 (should) + 动词原形 的虚拟结构 ，选原形 make 。
12单项选择B【易错陷阱】考查陈述语气 。当 insist 意为“坚持说、坚持认为（某事实已发生）”而不是“坚决要求”时，不用虚拟语气 。他坚持说他没偷钱包（事实） ，且发生在过去，用过去完成时，选 hadn't stolen 。
13单项选择C【易错陷阱】考查陈述语气 。当 suggest 意为“表明、暗示”而不是“建议”时，从句不用虚拟 。成功在微笑之前，用过去完成时，选 had succeeded 。
14单项选择B考查 wish 后的宾语从句 。表示对现在时间愿望的虚拟 ，结构为一般过去时，故选 learnt 。
15单项选择B考查 wish 后的宾语从句 。时间状语 last month 明确提示这是对过去未实现愿望的假设 ，虚拟结构固定使用 had + 过去分词 ，故选 had done 。
