#题型答案解析
01语法填空is主语 John 是第三人称单数，陈述客观事实用一般现在时，系动词用单数 is。
02语法填空is visiting考查就远原则。当主语后面跟有 along with 时，谓语动词的单复数取决于核心主语 The educator（单数），结合 now 用现在进行时。
03语法填空protectsclothing 是不可数集体名词，作主语时谓语动词用单数形式 protects。
04语法填空isthe number of + 复数名词 表示“……的数量”，核心主语是单数 number，谓语用单数 is。
05语法填空are sellinga number of + 复数名词 意为“许多……”，作主语谓语用复数；且 sell well 属于主动形式表达被动意义，结合 currently 用现在进行时 are selling。
06语法填空hasmany a + 单数名词 作主语，谓语动词用单数。结合 so far（现在完成时），填单三助动词 has。
07语法填空ismore than one + 单数名词 作主语，语法形式上跟从单数名词，谓语动词用单数 is。
08语法填空arescissors（剪刀）属于表示双数意义的复数名词，单独作主语时谓语动词用复数 are。
09语法填空belongs空格前有 This pair of 修饰，核心主语变成了单数 pair，一般现在时用单三形式 belongs。
10语法填空helps动名词短语 Reading English books aloud 作主语，谓语动词一律用单数形式 helps。
11单项选择B考查非谓语动词作后置定语。名词 information 与动词 provide 是被动关系，用过去分词形式 provided 作定语表示“在互联网上被提供的消息”。
12单项选择C考查倍数表达法。固定句型：倍数 + the + size/height/length + of + 比较对象。
13单项选择B考查主谓一致。many a + 单数名词 作主语时，谓语动词需用单数形式，故选单三形式 prefers。
14单项选择A考查就近原则。not only...but also... 连接并列主语，谓语动词与靠近它的单数主语 the child 保持一致，一般现在时用单三形式 wants。
15单项选择B考查主谓一致。本句的主语核心词是单数名词 taste（味道），而不是 food，因此谓语动词应用单数 is。
